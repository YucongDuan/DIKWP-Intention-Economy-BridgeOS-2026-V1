# DIKWP Intention Economy BridgeOS 2026 V1

DIKWP注意力经济到意图经济转型、意图授权与价值路由平台

A standalone, offline-first prototype for the transition from an attention economy to an intention economy.

## Core thesis

The attention economy monetizes exposure, clicks, time-on-feed, and behavioural prediction. The intention economy shifts the scarce object from attention to actionable user purpose. In a trustworthy version of the transition, user intent is clarified, authorized, matched, negotiated, executed, audited, and revocable.

## Workflow

Attention signal → Intent elicitation → Intent Contract → DIKWP content card → stakeholder intent space → semantic match → offer and route comparison → action gate → outcome ledger → revocation and learning → Intent Passport.

## Quick start

Open `index.html` in a modern browser. No server, database, account, API key, ad network, payment network, or external model is required.

Optional CLI:

```bash
python tools/run_intention_economy.py examples/sample_intent_passport.json --out outputs
```

## Boundary

This is a governance, learning, research, and demonstration prototype. It does not execute purchases, payments, bookings, brokerage transactions, public-service submissions, healthcare decisions, legal decisions, or any production actions.
